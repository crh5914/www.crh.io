    function getObjectUrl(file){
    var url = null;
    if(window.createObjectURL != undefined){ // basic case

    url = window.createObjectURL(file);

    }else if(window.URL != undefined){//IE and mozilla

    url = window.URL.createObjectURL(file);

    }else if(window.webkitURL != undefined){ //chrome and webkit

    url = window.webkitURL.createObjectURL(file);

    }
    return url;
    }

Send a QQ-email By Python

发表于 2017-12-02

用python发送qq邮件

目录

  • qq邮箱开启smtp服务
  • 编写python代码发送邮件

开启QQ邮箱smtp服务

如何开启QQ邮箱smtp服务

编写python代码发送邮件

环境

  • python3.5
  • email
  • smtplib

代码如下:

from email.mime.text import MIMEText
from email.header import Header
from smtplib import SMTP_SSL
host_server = 'smtp.qq.com'
sender = '*****@qq.com' 
password = '*****' # 开启smtp时拿到的密码
receiver = '*****@qq.com'
content = 'This is the first email sent by Python.'
title = 'A mail'
message = MIMEText(content,'plain','utf-8')
message['Subject'] = Header(title,'utf-8')
message['From'] = sender
message['To'] = receiver
smtp = SMTP_SSL(host_server)
try:
    smtp.login(sender,password)
    smtp.sendmail(sender,receiver, message.as_string())
except Exception as e:
    print('error',e)
finally:
    smtp.quit()
